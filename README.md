# Учебный проект — лабораторные работы

Кратко: в этом репозитории собраны лабораторные работы по моделированию предметных областей. Каждая лабораторная оформлена в отдельной папке и содержит описание предметной области, реализацию и тесты.

Краткое содержание лабораторных:
- `lr1` — лабораторная работа 1: «Университет» — реализовано на Go (POC: ontology, Kripke frames, интерактивный CLI);
- `lr2` — лабораторная работа 2: «Библиотека».
 - каждая лабораторная оформлена в своей папке (`lr1`, `lr2` и т.д.);
- для каждой лабораторной есть описание предметной области, Java-реализация, тесты и отчёт;
- для каждой лабораторной есть собственный `report.md`.

## Что вынесено из основной части
Файлы внутри папок (`report.md`, диаграммы, исходники и тесты) описывают реализацию и результаты для каждой лабораторной.
