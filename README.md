# Учебный проект — лабораторные работы по СТИС 2026г.

В этом репозитории собраны лабораторные работы по моделированию предметных областей, АТД. Каждая лабораторная  содержит описание предметной области, реализацию и тесты.

Краткое содержание лабораторных:
- `lr1` — лабораторная работа 1: «Университет» — реализовано на Go (POC: ontology, Kripke frames, интерактивный CLI);
- `lr2` — лабораторная работа 2: «Библиотека». Моделирование в среде SNEPS
