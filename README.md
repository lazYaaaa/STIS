# Учебный проект по SNePS

Этот репозиторий содержит набор лабораторных работ по моделированию предметных областей в SNePS.

## Что здесь лежит
- `lr1` — отдельная папка под первую лабораторную работу.
- `lr2` — лабораторная работа 2, модель предметной области "Библиотека".
- `lr3` — лабораторная работа 3, модель предметной области "Университет".
- `sneps` — локальная поставка SNePS и JavaSnepsAPI для запуска и проверки.

## Как это используется
- Основной рабочий запуск описан в `RUN_ALL.md`.
- Отчёты по каждой лабораторной лежат внутри соответствующих папок `lr1`, `lr2` и `lr3`.
- Для воспроизведения результатов используется контейнерный запуск и JavaSnepsAPI.

## Примечание
Папки `bdz1` и `bdz2`, а также PDF-условия заданий не входят в основную рабочую часть и игнорируются через `.gitignore`.
