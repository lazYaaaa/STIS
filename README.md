# Учебный проект по SNePS

Этот репозиторий собран для лабораторных работ по моделированию предметных областей в SNePS. Здесь важно не просто хранение файлов, а сама схема работы: выделить понятия, задать иерархию, описать экземпляры и проверить, как система делает выводы.

## Условия лабораторных
- выбрать предметную область с понятиями и связями;
- построить иерархию `ISA`;
- задать `instance-of` для конкретных объектов;
- проверить транзитивность и выводы по типам;
- подготовить тестовые факты и запросы;
- оформить краткий отчёт и ER-диаграмму.

## Что реализовано
- `lr2` - лабораторная работа 2 по теме «Библиотека»;
- `lr3` - лабораторная работа 3 по теме «Университет»;
- `lr1` - отдельная папка под первую лабораторную работу;
- для каждой лабораторной есть код, тесты и отчёт в своей папке;
- запуск и проверка выполняются через контейнер и JavaSnepsAPI.

## Как этим пользоваться
- основной сценарий запуска описан в `RUN_ALL.md`;
- для каждой лабораторной есть собственный `report.md`;
- рабочий путь воспроизведения результатов построен вокруг локальной поставки SNePS и JavaSnepsAPI.

## Что вынесено из основной части
- `bdz1` и `bdz2` не относятся к основным лабораторным;
- PDF-условия заданий тоже не входят в рабочую часть репозитория;
- всё это игнорируется через `.gitignore`.

## Итог
Это учебный репозиторий под SNePS, где каждая лабораторная оформлена как отдельный мини-проект с моделью, тестами и отчётом.
