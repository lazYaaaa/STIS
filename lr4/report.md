# Отчет по лабораторной работе №4

## Цель работы
Реализовать `LazySequence<T>`, семейство `Stream<T>` и задачу синхронизации временных потоков (вариант 5.1).

## Что сделано
- Реализованы `Sequence<T>`, `VectorSequence<T>`, `LazySequence<T>`.
- Реализованы `ReadOnlyStream<T>` и `WriteOnlyStream<T>` с файловым режимом.
- Реализован `SynchronizeStreams<T>` с окном `tau`.
- Добавлены CLI и встроенные тесты Catch2.
- Добавлен стресс-раннер с сохранением результатов и построением графика.

## Таблица соответствия требованиям
| Требование | Статус | Примечание |
|---|---|---|
| `LazySequence<T>` | выполнено | Реализован ленивый доступ с мемоизацией |
| Семейство `Stream<T>` | выполнено | Read/Write, поддержка памяти и файлов |
| Синхронизация потоков | выполнено | `SynchronizeStreams<T>` с окном `tau` |
| CLI для ручной проверки | выполнено | Меню демонстраций и стресс-тестов |
| Модульные тесты | выполнено | Catch2, встроенный раннер |
| Файловый режим | выполнено | чтение/запись потоков |
| Нагрузочные измерения | выполнено | стресс-раннер с сохранением результатов |

## Вывод
Поставленные требования выполнены. Реализованы АТД последовательностей, семейство потоков, синхронизация и проверка через CLI и тесты.
