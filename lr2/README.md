Предметная область
- Библиотека.
- Понятия: `Person`, `Reader`, `AdultReader`, `ChildReader`, `Librarian`, `Book`.
- Связи: `borrows` и `recommends`.
- Иерархия: `AdultReader` ISA `Reader`, `ChildReader` ISA `Reader`, `Reader` ISA `Person`, `Librarian` ISA `Person`, `Book` ISA `Item`.
